# Quiz-para-Progamadores
Quiz interativo para programadores desenvolvido com HTML5 e CSS3, com diferentes tipos de perguntas para praticar formulários e estruturação de páginas web.

Sobre o projeto

O quiz contém diferentes tipos de perguntas relacionadas à programação, abordando conceitos básicos como HTML, linguagens de programação e estrutura da web.

O projeto foi desenvolvido como parte dos meus estudos em desenvolvimento front-end, focando no fortalecimento da base antes de avançar para tecnologias mais complexas.

Tecnologias utilizadas:

- HTML5;
- CSS3;
-
Funcionalidades
O quiz inclui:

- Perguntas de múltipla escolha;
- Campo de texto;
- Campo de senha;
- Seleção de data;
- Checkbox (múltipla seleção)
- Upload de arquivo
- Menu suspenso (select)
- Exibição de respostas com details e summary
- Tabela de pontuação;
- Formulário de feedback.

Objetivo

Praticar:

- Estruturação semântica com HTML;
- Criação e organização de formulários;
- Estilização com CSS;
- Uso breve de Flexbox;
- Organização visual e boas práticas de layout.

Aprendizados:

Durante o desenvolvimento deste projeto, reforcei conceitos importantes da base do desenvolvimento web, como:

- Uso correto de inputs e seus tipos;
- Organização de conteúdo em seções
- Estilização consistente com CSS
- Separação entre estrutura (HTML) e estilo (CSS)
